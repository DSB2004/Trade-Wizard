TradeWizard - Stock Data Website

    TradeWizard is a prototype stock data website designed for analysis, providing users with near real-time stock data. This project offers valuable insights into stock market trends and facilitates data-driven decision-making for investors. While TradeWizard is currently a prototype, it has the potential to be further developed into a comprehensive platform for delivering real-time updates to users.

Caution:

    Avoid refreshing the chat page excessively to prevent unnecessary server load.
    Please send limited requests over API endpoints to ensure smooth functioning and avoid overwhelming the server.
    TradeWizard empowers users with timely and accurate stock information, enabling them to navigate the dynamic world of stock trading efficiently.



The server file is present in the directory. In order to use the website use need to run the development version on localhost and run the server separately.

To run frontend react: npm react
To run the server: node server.js

***This is just a prototype project and the data provided is near real-time update of current stock price please invest after consulting an expert in the stock market ***
